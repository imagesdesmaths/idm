Thickbox v3 - Documentation utilisateur


{{Thickbox_v3 :: fenêtres popup javascript}}


Ce plugin ajoute un effet [thickbox->http://codylindley.com/Javascript/257/thickbox-one-box-to-rule-them-all] sur les liens de type image (...) ou de class='thickbox'. Il fonctionne sur les squelettes disposant de la balise #INSERT_HEAD.

Il ajoute de nouvelles balises ainsi qu'un panel de configuration dans l'espace privé (cf. documentation).
